# Coturn TURN Server — Home Assistant Add-on Repository

This repository contains a Home Assistant add-on for running a 
[Coturn](https://github.com/coturn/coturn) TURN/STUN server.

## Installation

1. Go to **Settings → Add-ons → Add-on Store → ⋮ → Repositories**
2. Add this repository URL:
https://github.com/banditto9/coturn-addon
3. Find **Coturn TURN Server** under **Local add-ons** and click **Install**

## Add-ons

| Add-on | Description |
|--------|-------------|
| [Coturn TURN Server](coturn/) | A TURN/STUN server for NAT traversal, useful for WebRTC and SIP applications |
